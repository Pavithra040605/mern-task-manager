# MERN Task Manager 🚀

A simple full-stack Task Manager application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- MongoDB database integration
- REST API using Express.js
- React frontend with Axios

## Tech Stack

### Frontend
- React.js
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
